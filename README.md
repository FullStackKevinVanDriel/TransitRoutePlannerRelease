## Installation

### Free Version (Coming Soon)
```
WordPress Admin → Plugins → Add New → Search "Transit Route Planner"
```

### Pro Version
1. Purchase license at [vandromeda.com](https://vandromeda.com)
2. Download ZIP from email or customer portal
3. WordPress Admin → Plugins → Add New → Upload Plugin
4. Enter license key in Settings → Transit Route Planner

## Usage

Add to any page or post:
```
[transit_route_planner]
```
